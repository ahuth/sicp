# sicp

Exercises and notes from [Structure and Interpretation of Computer Programs](https://web.mit.edu/alexmv/6.037/sicp.pdf), first edition. In JavaScript.

## Usage

Install dependencies with:

```
npm install
```

Run the tests with:

```
npm test
```
